Detailed Description

CreaterObstacle class which extends the Obstacle class. It is meant to be used alongside BoulderObstacle. Both Obstacle types behave similarly except for how we draw the boundary box around it. For a creater, the boundary box is drawn arround the perimeter of the creater. For a bourder, we use it's height as the height box (meaning the box is vertical).

getterPoint() functions are called by other programs to outline boundary box surrounding obstacle

update() functions are called to update the bounds and height/depth of Obstacle depending on the new found point.

isSameAs() function is used to compare two obstacles and determe two Obstacles and determine if they should be combined or not
