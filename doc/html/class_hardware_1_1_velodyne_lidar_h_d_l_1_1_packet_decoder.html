Detailed Description

PacketDecoder class

The PacketDecoder class is intended to be used alonside the PacketDriver inside the VelodyneLidar wrapper class

DecodePacket(Byte[] packet) function is intended to be used to convert packet into a frame (explained in HDLFrame class)

GetLatestFrame() function is intended to be used to retreive the lastest HDLFrame with all of the provided decoded packets

Hardware.VelodyneLidarHDL.PacketDecoder.PacketDecoder ( boolean  generatePointCloud)

Constructor to PacketDecoder class. Input is used to indicate if the algorithm should take the packet and derive a point cloud 3D array or just store as distances (i.e. polar coordinates). Point cloud is only needed if no calibration data is avalible (flat surface was not sampled).

Parameters
